# Habit Goals — Трекер привычек и целей
 Управляйте целями. Формируйте привычки. Достигайте результатов.
 Персональный ассистент продуктивности, который помогает отслеживать прогресс, поддерживать серии и достигать поставленных целей каждый день.

Возможности
 Управление целями
  -Создание долгосрочных целей с датами начала и окончания
  -Статусы: Активна, Достигнута, Отменена
  -Поиск и фильтрация по статусу
  -Визуальный прогресс-бар
 Трекинг привычек
  -Ежедневные и еженедельные привычки
  -Привязка привычек к целям
  -Напоминания о выполнении
  -Подсчёт серий (streaks)
  -Быстрая отметка: выполнено или пропущено
 Аналитика и прогресс
  -Общий процент выполнимости
  -Максимальная серия дней
  -Прогресс по каждой привычке
  -Визуализация в реальном времени
 Профиль и активность
  -Личный кабинет с историей действий
  -Журнал активности (timeline)
  -Аватар с инициалами
 Авторизация
  -Регистрация новых пользователей
  -Вход по email и паролю
  -Роли: Пользователь и Администратор
 Админ-панель
  -Список всех пользователей
  -Журнал аудита действий системы
  -Сброс демо-данных
 Интерфейс
  -Тёмная и светлая темы с автоопределением
  -Glassmorphism-дизайн
  -Плавные анимации и переходы
  -Адаптивная вёрстка
  -Toast-уведомления
  -Модальные окна с валидацией форм
Технологический стек
  HTML5 — Семантическая разметка, dialog, template
  CSS3 — CSS-переменные, Grid, Flexbox, анимации, backdrop-filter
  JavaScript (ES6+) — Vanilla JS, без фреймворков
  LocalStorage — Персистентное хранение данных на клиенте
 Никаких зависимостей! Проект работает из коробки — нужен только браузер.

Архитектура
 Приложение построено на паттерне Single Page Application (SPA) с клиентским роутингом через hashchange.
 App
  state — Централизованное хранилище данных
  router() — Навигация по хешу (#dashboard, #goals...)
  renderAll() — Перерисовка всех view
  saveState() — Сохранение в localStorage
  audit() — Логирование действий
Сущности данных
  users — Пользователи системы
  goals — Цели
  habits — Привычки
  journal — Записи о выполнении
  notifications — Уведомления
  auditLog — Журнал аудита
Ключевые особенности реализации
  Валидация форм в реальном времени — кнопка сохранения блокируется до заполнения обязательных полей
  XSS-защита — метод escape() для безопасного рендеринга пользовательского ввода
  Reactive UI — любое изменение состояния автоматически перерисовывает интерфейс
  Темизация через CSS-переменные — мгновенное переключение без перезагрузки
  Аудит действий — каждое важное действие логируется с timestamp
